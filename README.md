# Java Currency Converter

This is a **Java console application** that allows users to convert currencies between:

- Philippine Peso (PHP)
- US Dollar (USD)
- Euro (EUR)
- Saudi Riyal (SAR)
- Indian Rupee (INR)

The program is written in Java using **basic concepts** like:

- `Scanner` for user input
- `if-else` statements for conversion logic
- `DecimalFormat` to format output to 3 decimal places

## Features

- Simple console-based interface
- Supports 5 currencies
- Displays formatted conversion results
- Handles invalid currency or conversion choices

## How to Run

1. Clone the repository:

```bash

2. Open the project in your favorite IDE (Eclipse, IntelliJ, NetBeans) or compile via terminal:
